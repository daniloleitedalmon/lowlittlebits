LowCost LittleBits

This is a project which main goal is to redesign Littlebits hardware in order to low its components costs.

For more information on Littlebits, please see http://littlebits.cc and its github repository https://github.com/littlebitselectronics/eagle-files

January 5th 2014
Currently the project has new designs for 2 bits. Resources are available in each bit's directory.
